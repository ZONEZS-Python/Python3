# 8.0.1 scrapy 初体验

##### 创建一个Scrapy 项目

```bash
# 通过命令 cd 到想要创建项目的目录中

$ scrapy startproject mingyan  

# 注释：scrapy startproject 为固定格式，可以创建一个mingyan的 Spider项目

```
